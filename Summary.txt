PROJECT OVERVIEW
================================================================================
A modern educational assessment platform for Australian students (Years 2-9)
focusing on NAPLAN and ICAS exam preparation.

Tech Stack: React + TypeScript + Vite + TailwindCSS + Supabase
Current Phase: Phase 1 - Foundation Complete ‚úÖ


QUICK START COMMANDS
================================================================================
npm install              # Install all dependencies
npm run dev             # Start development server (http://localhost:5173)
npm run build           # Build for production
npm run preview         # Preview production build
npm run lint            # Check code quality

SRC/STORE/ - STATE MANAGEMENT
--------------------------------------------------------------------------------

üìÑ src/store/index.ts
   - Zustand state management
   - Contains 3 stores:
   
   1. useAuthStore - User authentication state
      - user: User object or null
      - isAuthenticated: boolean
      - setUser(user)
      - logout()
   
   2. useExamStore - Exam session state
      - currentExamId: string
      - examStartTime: Date
      - answers: Record of question answers
      - flaggedQuestions: Set of flagged questions
      - saveAnswer(questionId, answer)
      - toggleFlag(questionId)
   
   3. useUIStore - UI state
      - isSidebarOpen: boolean
      - toggleSidebar()
   
   MODIFY: When adding new global state needs

COMMON MODIFICATIONS
--------------------

1. CHANGE COLORS
   ‚Üí Edit: tailwind.config.js
   ‚Üí Look for: theme.extend.colors
   ‚Üí Modify: primary, success, warning, danger color values

2. ADD NEW PAGE
   ‚Üí Create: src/pages/YourPage.tsx
   ‚Üí Add route: src/lib/router.tsx
   ‚Üí Add constant: src/data/constants/index.ts (ROUTES)

3. CHANGE APP NAME
   ‚Üí Edit: src/data/constants/index.ts (APP_NAME)
   ‚Üí Edit: index.html (<title>)
   ‚Üí Edit: package.json (name)

4. ADD NEW SUBJECT
   ‚Üí Edit: src/data/constants/index.ts
   ‚Üí Add to SUBJECTS object

   ‚Üí Update types in src/types/index.ts if needed

5. MODIFY NAVBAR
   ‚Üí Edit: src/components/layout/Navbar.tsx
   ‚Üí Add new links or buttons

6. ADD GLOBAL STYLE
   ‚Üí Edit: src/styles/index.css
   ‚Üí Add under @layer components or @layer utilities

7. ADD NEW STATE
   ‚Üí Edit: src/store/index.ts
